# myip 🌐
Check your IP settings with this command<br>
Basically it sends GET request to ipinfo.io

# encrypt 🔒
Encrypt your data with this command<br>
Using AES algo

# decrypt 🔓
Decrypt an AES string<br>
Using AES algo

# pokemon 🎱
Returns random pokemon and it's data

# cat 🐱
Reads text files

# chars_count 🅱️
Counts chars

# words_count 🆎
Counts words

# mkdir 📂
Creates directory

# touch 📄
Creates file
